Unidad 1 Actividad 06: Listas de Siglas

Siglas

OSI (Open Systems Interconnection)

Modelo conceptual que divide las funciones de red.

IP (Internet Protocol)

Protocolo que identifica dispositivos en una red mediante direcciones IP.

TCP (Transmission Control Protocol)

Protocolo de control de transmisión que garantiza la entrega confiable de datos.

UDP (User Datagram Protocol)

Protocolo que permite la transmisión rápida de datos sin establecer una conexión previa.

SSH (Secure Shell)

Protocolo de administración remota que permite acceso seguro a servidores.

FTP (File Transfer Protocol)

Protocolo utilizado para transferir archivos entre dispositivos en una red.

SMTP (Simple Mail Transfer Protocol)

Protocolo utilizado para el envío de correos electrónicos.

POP (Post Office Protocol)

Protocolo utilizado para recibir correos electrónicos desde un servidor remoto.

DHCP (Dynamic Host Configuration Protocol)

Protocolo que asigna automáticamente direcciones IP a dispositivos en una red.

DNS (Domain Name System)

Servicio que traduce nombres de dominio en direcciones IP.

HTTP (HyperText Transfer Protocol)

Protocolo que permite la transferencia de información en la web.

ARP (Address Resolution Protocol)

Protocolo que vincula direcciones IP con direcciones físicas MAC.

MAC (Media Access Control)

Dirección física única asignada a cada dispositivo de red.

LAN (Local Area Network)

Red de computadoras en un área geográfica limitada.
