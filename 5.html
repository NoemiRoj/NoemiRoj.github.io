Unidad 1 Actividad 05:: Tema (Listas)

Siglas

OSI
Open Systems Interconnection (Interconexión de Sistemas Abiertos). Es un modelo de referencia para los protocolos de comunicación de redes informáticas.
IP
Internet Protocol (Protocolo de Internet). Es una dirección única que identifica un dispositivo en Internet o en una red local.
TCP
Transmission Control Protocol (Protocolo de Control de Transmisión). Es el método de comunicación de datos por defecto entre distintos dispositivos a través de una red.
UDP
User Datagram Protocol. Permite la transmisión sin conexión de datagramas en redes basadas en IP.
SSH
Secure Shell. Protocolo de administración remota que permite a los usuarios controlar y modificar servidores remotos a través de Internet.
FTP
File Transfer Protocol (Protocolo de Transferencia de Archivos). Permite transferir archivos directamente de un dispositivo a otro.
SMTP
Simple Mail Transfer Protocol. Protocolo de red utilizado en el envío y recepción de correos electrónicos.
POP
Post Office Protocol. Establece una conexión entre un cliente y un servidor de correo electrónico para gestionar el envío de mensajes.
DHCP
Dynamic Host Configuration Protocol (Protocolo de Configuración Dinámica de Host). Protocolo de red que utiliza una arquitectura cliente-servidor para asignar direcciones IP automáticamente.
DNS
Domain Name System. Traduce los nombres de dominio a direcciones IP para que los navegadores puedan cargar los recursos de Internet.
HTTP
HyperText Transfer Protocol (Protocolo de Transferencia de Hipertexto). Es el protocolo de transmisión de información de la World Wide Web.
ARP
Address Resolution Protocol (Protocolo de Resolución de Direcciones). Vincula una dirección MAC con una dirección IP.
MAC
Media Access Control. Es la dirección física de un dispositivo de red, utilizada para identificarlo en redes cableadas e inalámbricas.
LAN
Local Area Network (Red de Área Local). Conjunto de dispositivos conectados en un área geográfica limitada para compartir recursos y comunicación.
